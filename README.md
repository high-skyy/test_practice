## 재귀로 풀 경우 깊이 때문에 런타임 에러가 발생할 수 있음
> 아래와 같은 코드로 이를 해결할 수 있다.
```
import sys
sys.setrecursionlimit(10**6)
```

## 데이터 크기 제한 & 시간 복잡도
- n <= 1,000,000        :       O(n) or O(n*log(n))
- n <= 10,000           :             O(n**2)
- n <= 500              :             O(n**3)
