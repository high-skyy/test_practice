# 누적합(prefix sum)

2차원 배열이 있다고 가정을 하자.  
(1,3) ~ (4,5) 까지 새로운 값을 추가해 주는 경우  
(1,3) 해당 추가하는 값을 넣어주고  
(1,6), (5,3) 에 해당 하는 값 - 넣어주고  
(5,6)에 해당하는 값 + 넣어준다.  
행 한번 다 연속적으로 더하고  
열 한번 다 연속적으로 더한다.  
피보나치 수열 처럼
